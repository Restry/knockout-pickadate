Knockout Pickadate

Knockout Pickadate Demo

Simple example

Use the pickadate binding on any input[type="text"] element.

            window.view = {
              date: ko.observable()
            };

            ko.applyBindings(view);
          
            <input type='text' data-bind='pickadate: date'>
            <p>
              <strong>Value:</strong>
              <span data-bind='text: date'></span>
            </p>

Options:

You can pass in the following options as the pickadate_options binding:

Pickadate
See Pickadate for a list of options available. All options specified in pickadate_options will be passed to pickadate.
calendar_addon
true to enable the calendar button addon. Default: false
update_as_date
true to make the binding update the observable with a JavaScript Date object instead of text. Default: false
Note: Requires Bootstrap 3 and Font Awesome
